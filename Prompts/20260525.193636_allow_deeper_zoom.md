# Implementation Plan - Allow Zooming in Past Initial Limits

To ensure users can zoom in past the initial 20m x 20m track bounding box limits (which corresponds to zoom levels ~22-23 on standard devices), we will configure the MapLibre Map camera settings to allow a maximum zoom preference of 24.0.

## Proposed Changes

### `MapActivity.java`

1. **Explicitly Set Max Zoom Preference**:
   In `onMapReady(MapLibreMap mapLibreMap)`, we will configure the maximum allowed zoom preference on the `mapLibreMap` object:
   ```java
   mapLibreMap.setMaxZoomPreference(24.0);
   ```
   This ensures that the user is not restricted by MapLibre's default zoom constraints (which can restrict manual zoom depending on styling/cache defaults) and can zoom in as closely as needed to view high-resolution paths and single coordinate points.
