# Implementation Plan for Adding Back Arrows to Secondary Screens

**User Prompt:**
> Like the 'Tracks List' screen, have a left arrow to close screen on 'Settings', 'Help' and 'About' screen also.

1. **Investigate the Problem**:
    * `SettingsActivity`, `HelpActivity`, and `AboutActivity` currently don't show the back arrow because the application theme (`Theme.GPSLocationLogger`) is likely a `NoActionBar` theme.
    * They don't have a `Toolbar` element defined in their layout files (`activity_settings.xml`, `activity_help.xml`, `activity_about.xml`).
    * Without a `Toolbar` added and set via `setSupportActionBar()`, `getSupportActionBar()` returns null, and `setDisplayHomeAsUpEnabled(true)` does nothing.

2. **Implementation Steps**:
    * **SettingsActivity**:
        * Update `activity_settings.xml` to wrap the `ScrollView` in a `LinearLayout` and add a `Toolbar` at the top.
        * Update `SettingsActivity.java` to call `setSupportActionBar(toolbar)`.
        * Handle `onOptionsItemSelected` for `android.R.id.home` to call `finish()`.
    * **HelpActivity**:
        * Update `activity_help.xml` to include a `Toolbar`.
        * Update `HelpActivity.java` to set up the toolbar and handle the back action.
    * **AboutActivity**:
        * Update `activity_about.xml` to include a `Toolbar`.
        * Update `AboutActivity.java` to set up the toolbar and handle the back action.
