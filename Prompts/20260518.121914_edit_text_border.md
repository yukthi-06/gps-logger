# Implementation Plan for Map Style URL Text Field Outlined Border

**User Prompt:**
> Map Style URL text field doesnt look like a text field. Add a border to the field

1. **Investigate the Problem**:
    * Currently, the Map Style URL `EditText` inside the card on the settings page has no defined background or border.
    * In standard Android themes, this results in the field having either no borders or just a flat line, making it hard to identify as an interactive text input.
    * Wrapping it with a beautiful, high-contrast, rounded outlined border with smooth visual states (e.g. highlight on focus) gives a premium, professional UI.

2. **Implementation Steps**:
    * **`bg_edit_text.xml`**: Create a state-list selector shape drawable that renders a subtle grey border (`#D0D4DC`) on a clean greyish-white background (`#F8F9FA`) by default, and transforms into a distinct Navy blue border (`#0F3460`) with a white background when focused/active.
    * **`activity_settings.xml`**: Apply this new `@drawable/bg_edit_text` background directly to `etMapStyleUrl` and insert a clean top margin to separate it from the divider.
