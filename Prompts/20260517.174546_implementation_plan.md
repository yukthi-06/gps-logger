# Implementation Plan for Tracks List Feature

**User Prompt:**
> on the left side menu, create a new menu item by name 'Tracks List'. Clicking on this should launch a new screen which shows list of tracks from the saved folder. Since for each track there are 3 files(gpx, kml, json), only one item should be shown in the tracks list.



1. **Investigate Current State**:
    * Identify the menu resource file for the left-side menu (e.g., `nav_menu.xml` or `activity_main_drawer.xml`).
    * Identify where the navigation drawer item clicks are handled (e.g., `MainActivity.java`).
    * Determine the exact path of the saved folder where tracks are stored (from previous tasks, it's likely `/sdcard/Vypeensoft/GPS_Location_Logger`).

2. **Add Menu Item**:
    * Add a new item "Tracks List" to the navigation drawer menu resource.
    * Add the corresponding string resource to `strings.xml`.

3. **Create TracksListActivity**:
    * Create a new Activity class `TracksListActivity.java`.
    * Create a layout file `activity_tracks_list.xml` containing a `RecyclerView` or `ListView` and an empty state `TextView`.
    * Declare `TracksListActivity` in `AndroidManifest.xml`.

4. **Implement File Reading and Grouping**:
    * In `TracksListActivity`, read the contents of the save directory.
    * Extract file names and group them by base name (removing `.gpx`, `.kml`, `.json` extensions).
    * Create a list of unique track names.

5. **Display Tracks**:
    * Create a simple adapter (e.g., `ArrayAdapter` or custom `RecyclerView.Adapter`) to display the unique track names.
    * Set the adapter to the list view in the activity.

6. **Handle Navigation**:
    * Update the navigation item selection listener in `MainActivity.java` to start `TracksListActivity` when "Tracks List" is clicked.
