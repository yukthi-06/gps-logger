# Implementation Plan for Default List Item Action: Show Route on Map

**User Prompt:**
> default action on track list item should be "Show route on map"

1. **Investigate the Problem**:
    * Currently, inside the tracks list recyclerview adapter (`TrackAdapter.java`), clicking on individual buttons (Map, Share, Delete) executes their respective actions.
    * However, clicking on the overall list item row/card itself does nothing because `holder.itemView` has no click listener attached.
    * To deliver a smooth, natural, and modern mobile user experience, clicking anywhere on the track item card itself should trigger the map action.

2. **Implementation Steps**:
    * **`TrackAdapter.java`**: Add a click listener to `holder.itemView` inside `onBindViewHolder()` that routes directly to the listener's `onMapClick(trackItem)` callback.
