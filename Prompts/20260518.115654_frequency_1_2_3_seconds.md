# Implementation Plan for Fast Logging Frequency (1, 2, 3 Seconds)

**User Prompt:**
> Provide logging frequency of 1, 2, 3 seconds also in settings

1. **Investigate the Requirements**:
    * The user wants logging frequencies of 1 second, 2 seconds, and 3 seconds available in settings.
    * Currently, the lowest frequency available is 5 seconds.
    * The `RadioGroup` for logging frequency inside `activity_settings.xml` maps to the parallel arrays `RADIO_IDS`, `INTERVALS_MS`, and `INTERVAL_LABELS` in `SettingsActivity.java`.
    * The background `LocationService.java` automatically queries `SettingsActivity.KEY_INTERVAL_MS` to define the Fused Location Provider's `LocationRequest` update intervals.

2. **Implementation Steps**:
    * **`strings.xml`**: Define string resources `freq_1s`, `freq_2s`, and `freq_3s`.
    * **`activity_settings.xml`**: Insert RadioButtons `rb1s`, `rb2s`, and `rb3s` at the top of the `rgFrequency` RadioGroup.
    * **`SettingsActivity.java`**: Update the parallel arrays (`RADIO_IDS`, `INTERVALS_MS`, `INTERVAL_LABELS`) to map these three new radio button IDs to 1_000ms, 2_000ms, and 3_000ms.
