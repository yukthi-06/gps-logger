# Implementation Plan - Settings JSON Synchronization and Tracks Path Redirection

We will implement support for saving settings values as a JSON file and redirect track files to a dedicated `tracks` subdirectory.

## Objectives
1. **Tracks Path Redirection**: Move the default track storage directory from `/sdcard/Vypeensoft/GPS_Location_Logger/` to `/sdcard/Vypeensoft/GPS_Location_Logger/tracks/` across all activities and helper classes.
2. **Settings JSON Synchronization**:
   - Create a helper utility `SettingsHelper.java` to synchronize `SharedPreferences` with the external file `/sdcard/Vypeensoft/GPS_Location_Logger/settings/settings.json`.
   - Automatically **load** external settings into `SharedPreferences` when the application starts or options are opened.
   - Automatically **save** settings to the JSON file on external storage whenever a setting is updated in options.

## Solution Design

### 1. Tracks Path Redirection
Update the `GPS_FOLDER` constant across all relevant files from `"Vypeensoft/GPS_Location_Logger"` to `"Vypeensoft/GPS_Location_Logger/tracks"`.
Impacted files:
- `MainActivity.java`
- `MapActivity.java`
- `SharePointHelper.java`
- `TracksListActivity.java`
- `strings.xml` (save path label string)

### 2. Settings JSON Synchronization (`SettingsHelper.java`)
We will create a new class `SettingsHelper.java` containing:
- `saveSettingsToJson(Context context, SharedPreferences prefs)`: Serializes current settings to a JSON file on disk at `/sdcard/Vypeensoft/GPS_Location_Logger/settings/settings.json`.
- `loadSettingsFromJson(Context context, SharedPreferences prefs)`: If the settings JSON file exists, reads it and populates `SharedPreferences` with the saved values, allowing seamless integration.

This helper will be integrated into:
- `MainActivity.onCreate()`: Load settings from JSON into `SharedPreferences` right away so the service/activity has access to external configurations.
- `SettingsActivity.onCreate()`: Load settings from JSON into `SharedPreferences` before initializing preference controls.
- `SettingsActivity.java` listeners: Call `SettingsHelper.saveSettingsToJson(this, prefs)` whenever the frequency spinner, formats check boxes, or map style URL text is changed.
