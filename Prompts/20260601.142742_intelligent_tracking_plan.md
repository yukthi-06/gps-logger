# Implementation Plan - Intelligent Location Tracking

We will implement a premium "Intelligent Tracking" feature that dynamically skips recording consecutive GPS fixes if they are within a configurable distance threshold.

## Objectives
1. **Settings Keys**:
   - `KEY_INTELLIGENT_TRACKING` = `"intelligent_tracking"` (boolean, defaults to `false`).
   - `KEY_MIN_DISTANCE_METERS` = `"min_distance_meters"` (float, defaults to `5.0f`).
2. **Options Screen UI**:
   - Add a clean card section with a toggle switch/checkbox for "Enable Intelligent Tracking".
   - Underneath, add a high-fidelity dropdown Spinner that lets users configure the distance threshold (2m, 5m, 10m, 20m, 50m).
   - Show/hide or enable/disable the distance threshold selector depending on whether intelligent tracking is enabled.
3. **Background Filtration (`LocationService.java`)**:
   - Retrieve preference flags.
   - If intelligent tracking is active, calculate the distance between the incoming coordinate and the last logged coordinate using `Location.distanceBetween`.
   - If the distance is strictly less than the selected threshold, skip logging and log the event.
4. **Serialization (`SettingsHelper.java`)**:
   - Include these keys in `settings.json` so that external config modifications persist automatically.

## Solution Design

### 1. Settings Activity Additions
- Declare keys in `SettingsActivity.java`:
  ```java
  public static final String KEY_INTELLIGENT_TRACKING = "intelligent_tracking";
  public static final String KEY_MIN_DISTANCE_METERS = "min_distance_meters";
  public static final float DEFAULT_MIN_DISTANCE_METERS = 5.0f;
  ```
- Declare arrays for distance choices:
  ```java
  private static final float[] DISTANCES_M = { 2.0f, 5.0f, 10.0f, 20.0f, 50.0f };
  private static final String[] DISTANCE_LABELS = { "2 meters", "5 meters (Default)", "10 meters", "20 meters", "50 meters" };
  ```
- Toggle view states dynamically: if the checkbox is checked, make the distance spinner visible, otherwise hide/disable it to save screen space.

### 2. Location Service Additions
- Inside `LocationService.handleNewLocation(Location location)`:
  - If `locationRecords` is not empty, get the last logged coordinate.
  - Calculate distance via `Location.distanceBetween`.
  - If `intelligent_tracking` is true and `distance < min_distance_meters`, log a message and exit immediately.
